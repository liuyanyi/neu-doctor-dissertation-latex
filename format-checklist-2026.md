# 东北大学博士学位论文格式检查清单（2026 版）

来源：`2026-原始版本/2-学位论文格式（博士）（页眉部分只保留第几章的信息，2026版）.docx`

说明：本清单先整理 Word 文档中可用于核对 LaTeX 模板的格式要求。虽然文件名标注“页眉部分只保留第几章的信息”，但 Word 正文第 3 节写明页眉为“章号章题”，且文档后部示例页眉显示为“第 1 章 · 引言”。后续检查时按“章号 + 章题”执行。

## 1. 论文组成

### 1.1 前头部分

- [ ] 包含封面。
- [ ] 包含扉页/题名页，且有中文、英文两种。
- [ ] 包含声明/独创性声明。
- [ ] 包含中文摘要和英文摘要。
- [ ] 包含目录。
- [ ] 必要时包含插图和附表清单。
- [ ] 必要时包含缩略字、缩写词、符号、单位表。
- [ ] 必要时包含名词术语注释表。

### 1.2 主体部分

- [ ] 包含绪论/前言/引言/绪言。
- [ ] 包含正文。
- [ ] 包含讨论、结论和建议。

### 1.3 结尾部分

- [ ] 包含参考文献。
- [ ] 包含致谢。
- [ ] 包含攻读博士学位期间取得的学术成果。
- [ ] 必要时包含作者从事科学研究和学习经历的简历。
- [ ] 必要时包含可供参考的文献题录。
- [ ] 必要时包含索引。

## 2. 页面与版式

- [ ] 纸张为标准 A4 复印纸，尺寸 `210 mm x 297 mm`。
- [ ] 版芯/打印尺寸为 `160 mm x 247 mm`，不包括页眉行、页码行。
- [ ] 正文字体字号全文统一，为小四号宋体。
- [ ] 每页 30 至 35 行。
- [ ] 每行 35 至 38 字。
- [ ] 采用双面打印印刷。
- [ ] 沿长边装订。
- [ ] 封面采用东北大学研究生博士学位论文标准封面，双 A4。

## 3. 页码与页眉

- [ ] 正文前部分（包括声明、摘要、Abstract、目录、常见缩略语表等）使用罗马数字页码编号。
- [ ] 正文从第 1 章开始使用阿拉伯数字页码，并从 `1` 开始编号。
- [ ] 页码字号与正文字体相同。
- [ ] 页码位于页底居中。
- [ ] 页码数字两侧使用圆点或一字横线修饰，例如 `·3·` 或 `-3-`。
- [ ] 自摘要页起加页眉。
- [ ] 页眉线可用单线或双线（二等线、文武线）。
- [ ] 页眉文字为 5 号楷体。
- [ ] 页眉文字居中。
- [ ] 正文页眉保留章号和章题，例如 `第 1 章 · 引言`。
- [ ] 前置部分页眉按对应部分标题显示，例如 `摘要`、`Abstract`、`目录`。

## 4. 标题层级与编号

- [ ] 正文按章、条、款、项分级。
- [ ] 章、条、款、项之间使用阿拉伯数字编号。
- [ ] 不同级编号之间使用半角实心下圆点 `.` 分隔。
- [ ] 最末级编号之后不加点。
- [ ] 分级编号最多到第四级。
- [ ] 第四级之后如需再分，使用 `（1）`、`（2）` 或 `（a）`、`（b）` 等形式。
- [ ] 第一级标题（章）为二号黑体。
- [ ] 第一级标题居中，占 3 行。
- [ ] 第一级标题格式示例为 `第1章 XXX`。
- [ ] 第二级标题（条）为三号黑体。
- [ ] 第二级标题居左，占 2 行。
- [ ] 第二级标题格式示例为 `1.1 XXXXXX`。
- [ ] 第三级标题（款）为四号黑体。
- [ ] 第三级标题居左，占 2 行。
- [ ] 第三级标题格式示例为 `1.1.1 XXXXXX`。
- [ ] 第四级标题（项）为小四号黑体。
- [ ] 第四级标题居左，占 1 行。
- [ ] 第四级标题格式示例为 `1.1.1.1 XXXXXX`。
- [ ] 摘要、目录、参考文献、致谢、攻读博士学位期间取得的学术成果、个人简历等标题按第一级标题排版。

## 5. 正文文字

- [ ] 中文正文为小四号宋体。
- [ ] 外文和数字字号与同行汉字字号相同。
- [ ] 外文和数字使用 Times New Roman 或相近字体。

## 6. 插图

- [ ] 插图可包括图解、示意图、构造图、曲线图、框图、流程图、布置图、地图、照片、图版等。
- [ ] 插图注明项包括图号、图题、图例。
- [ ] 图号按章序号编码，例如 `图2.1` 表示第 2 章第 1 图。
- [ ] 图号与图题文字之间空一字。
- [ ] 图号与图题置于图的正下方。
- [ ] 图题使用 5 号字。
- [ ] 图题字体可用宋体，且全文统一。
- [ ] 图中标注、符号、文字字号不大于图题字号。

## 7. 表格

- [ ] 表格数据依序竖排。
- [ ] 表格内容和项目由左至右横读。
- [ ] 表格通版排版。
- [ ] 表号按章序号编码，例如 `表2.1` 表示第 2 章第 1 表。
- [ ] 表应有表题。
- [ ] 表题与表号之间空 1 至 2 字。
- [ ] 表题置于表的上方居中。
- [ ] 表题使用 5 号宋体。
- [ ] 表题格式全文统一。
- [ ] 表中内容和项目字号不大于图题字号。
- [ ] 如使用中英文双语表题，按示例中文表题在前、英文表题在后，例如 `表2.1 ...`、`Table 2.1 ...`。

## 8. 公式

- [ ] 公式包括数学、物理和化学公式。
- [ ] 正文中引用的公式、算式或方程式可按章序号用阿拉伯数字编号。
- [ ] 公式编号格式为 `（2.1）`，表示第 2 章第 1 式。
- [ ] 公式一般单行居中排版。
- [ ] 公式与上下文分开。
- [ ] 式号与公式同行居右排版。

## 9. 附录

- [ ] 附录中的图、表、公式、参考文献等另行编排序号，与正文分开。
- [ ] 附录序号一律使用阿拉伯数字。
- [ ] 附录图、表、公式等编号在数码前加附录序码，例如 `图A.1`、`式（B.3）`。

## 10. 计量单位

- [ ] 全文采用《中华人民共和国法定计量单位》。
- [ ] 遵照《中华人民共和国法定计量单位使用方法》执行。
- [ ] 量、单位和符号遵循国家标准 `GB3100-82`、`GB3101-82`、`GB3102/1-13-82` 等规定。
- [ ] 单位名称和符号可采用国际通用符号。
- [ ] 单位名称和符号也可采用中文名称。
- [ ] 单位书写方式全文统一，不混用国际通用符号和中文名称。

## 11. 参考文献

- [ ] 参考文献采用顺序号编号体系。
- [ ] 专著格式为：`[序号] 编著者. 书名[M]. 出版地：出版社，年代，起止页码.`
- [ ] 期刊论文格式为：`[序号] 作者. 论文名称[J]. 期刊名称，年度，卷（期）：起止页码.`
- [ ] 学位论文格式为：`[序号] 作者. 学位论文名称[D]. 发表地：学位授予单位，年度.`

## 12. 攻读博士学位期间取得的学术成果

- [ ] 期刊成果格式为：`[序号] 作者. 论文名称[J]. 期刊名称，年度，卷（期）：起止页码. （检索情况）（对应论文章节）`
- [ ] 专利成果格式为：`[序号] 专利申请者. 专利题名：专利国别，专利号[P]. 发布日期. （对应论文章节）`
- [ ] 成果条目注明检索情况。
- [ ] 成果条目注明对应论文章节。
- [ ] 双盲评审版学位论文隐去所有作者/申请者姓名。
- [ ] 双盲评审版仅标注作者排序或申请者排序，例如 `第一作者`、`第二作者`、`第二排序`。

## 13. 封面与题名页

- [ ] 中文封面包含分类号。
- [ ] 中文封面包含密级。
- [ ] 中文封面包含 UDC。
- [ ] 中文封面显示 `学 位 论 文`。
- [ ] 中文封面包含中文论文题名。
- [ ] 中文封面包含作者姓名。
- [ ] 中文封面包含作者学号。
- [ ] 中文封面包含指导教师姓名、职称及单位。
- [ ] 中文封面包含申请学位级别。
- [ ] 中文封面包含学科类别。
- [ ] 中文封面包含学科专业名称。
- [ ] 中文封面包含论文提交日期。
- [ ] 中文封面包含论文答辩日期。
- [ ] 中文封面包含学位授予日期。
- [ ] 中文封面包含答辩委员会主席。
- [ ] 中文封面包含学校名称 `东北大学`。
- [ ] 中文封面包含日期。
- [ ] 英文题名页包含 `A Dissertation in ...`。
- [ ] 英文题名页包含英文论文题名。
- [ ] 英文题名页包含作者姓名。
- [ ] 英文题名页包含导师信息。
- [ ] 英文题名页包含 `Northeastern University`。
- [ ] 英文题名页包含日期。

## 14. 声明与授权

- [ ] 包含独创性声明标题。
- [ ] 包含独创性声明正文。
- [ ] 包含学位论文作者签名栏。
- [ ] 包含日期栏。
- [ ] 包含学位论文版权使用授权书标题。
- [ ] 包含版权使用授权书正文。
- [ ] 包含网上交流时间选项：半年、一年、一年半、两年。
- [ ] 包含学位论文作者签名栏。
- [ ] 包含导师签名栏。
- [ ] 包含双方签字日期栏。

## 15. 摘要与关键词

- [ ] 中文摘要标题为 `摘  要`，按第一级标题排版。
- [ ] 英文摘要标题为 `Abstract`，按第一级标题排版。
- [ ] 中文摘要正文使用正文格式。
- [ ] 英文摘要正文使用正文格式。
- [ ] 中文关键词以 `关键词：` 开头。
- [ ] 中文关键词之间使用分号分隔。
- [ ] 英文关键词以 `Key words:` 开头。
- [ ] 英文关键词之间使用分号分隔。

## 16. 目录

- [ ] 目录标题为 `目  录`，按第一级标题排版。
- [ ] 目录包含声明。
- [ ] 目录包含中文摘要。
- [ ] 目录包含英文摘要。
- [ ] 目录包含正文各章。
- [ ] 目录包含正文各级标题。
- [ ] 目录条目使用引导点连接标题与页码。
- [ ] 目录页码与正文页码对应。
